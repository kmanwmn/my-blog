## 后端 API 文档

### 基础地址
- 开发: `http://127.0.0.1:5000`
- 生产（直连）: `https://upload.sensing-origin.cn:8080`
- 生产（Vercel 代理，仅小请求）: `https://sensing-origin.cn/api/*`

### 视频管理

#### 获取视频列表
```
GET /api/blog-videos
```
返回所有视频，按 sort 降序。

#### 获取单个视频
```
GET /api/blog-videos/:id
```

#### 获取可用文件列表（已上传未入库）
```
GET /api/blog-videos/available-files
```
返回 `[{ "filename": "blog_xxx.mp4" }]`，按修改时间倒序。

#### 新建视频（需登录）
```
POST /api/blog-videos
Content-Type: application/json
Authorization: Bearer <token>

{
    "title_zh": "视频标题",
    "desc_zh": "视频描述",
    "filename": "blog_xxx.mp4",
    "sort": 0,
    "tags": "SAR,雷达成像,算法"
}
```
不再需要 duration/cover_gradient/cover_icon 字段，前端已移除。

#### 编辑视频（需登录）
```
PUT /api/blog-videos/:id
```

#### 删除视频（需登录）
```
DELETE /api/blog-videos/:id
```

### 视频上传（需登录）

```
POST /api/upload/blog-video
Content-Type: application/octet-stream
Authorization: Bearer <token>
X-Filename: demo.mp4
```

**特性：**
- 支持分片上传，URL 参数 `?chunk=0&total=10`
- 文件自动添加 `blog_` 前缀保存（如 `blog_demo.mp4`）
- 上传完成自动用 ffmpeg 截取第 5 秒画面作封面
- 后端记录已上传的分片（`<filename>.chunks` 文件），支持断点续传

#### 查询上传进度（续传用）
```
GET /api/upload/blog-video/status?filename=blog_xxx.mp4
```
返回已完成的分片列表 `{ "filename": "...", "chunks": [0,1,2,3], "total": 4 }`。

### 视频播放

```
GET /vod/blog_xxx.mp4
```
视频文件直连，支持 Range 请求（拖动进度条）。

### 视频封面

```
GET /covers/blog_xxx.mp4.jpg
```
ffmpeg 截取的封面图，视频卡片默认使用此图，不存在时回退到渐变色。

### 评论管理

#### 获取评论列表
```
GET /api/blog-comments?video_id=1
```
返回该视频的所有评论，按时间升序，包含 `parent_id` 用于展示回复关系。

#### 发表评论（无需登录）
```
POST /api/blog-comments
Content-Type: application/json

{
    "video_id": 1,
    "nickname": "访客名",
    "email": "可选@邮箱.com",
    "content": "评论内容",
    "parent_id": 0,
    "honeypot": ""
}
```
- `honeypot` 蜜罐防 spam，被填时后端静默返回成功但不保存
- `parent_id` 回复时填被回复评论的 ID

#### 删除评论（需登录）
```
DELETE /api/blog-comments/:id
```

### 认证

#### 管理员登录
```
POST /api/auth
Content-Type: application/json

{"username": "liyuchen", "password": "colin123"}
```
返回 token，后续请求通过 `Authorization: Bearer <token>` 传递。

## 数据库表结构

### t_BlogVideos

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT AUTO_INCREMENT | 主键 |
| sort | INT | 排序 |
| title_zh | VARCHAR(200) | 标题 |
| desc_zh | TEXT | 描述 |
| filename | VARCHAR(280) | 文件名（自动带 blog_ 前缀） |
| tags | VARCHAR(200) | 标签，逗号分隔 |
| duration | VARCHAR(20) | 时长（ffmpeg 自动计算） |
| cover_gradient | VARCHAR(200) | 封面渐变（不再使用） |
| cover_icon | VARCHAR(20) | 封面图标（不再使用） |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### t_BlogComments

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT AUTO_INCREMENT | 主键 |
| video_id | INT | 关联视频 ID |
| parent_id | INT | 回复的父评论 ID |
| nickname | VARCHAR(50) | 昵称 |
| email | VARCHAR(200) | 邮箱（可选） |
| content | TEXT | 评论内容 |
| created_at | TIMESTAMP | 创建时间 |

## 前端特性

- **暗色/浅色主题** — 默认浅色，右上角切换，存入 localStorage
- **响应式布局** — 手机/平板/桌面自适应
- **管理员面板** — Ctrl+` 打开，支持视频 CRUD + 分片上传
- **新增视频** — 下拉框选已上传的文件，自动填入时长，支持自定义标签
- **封面图** — ffmpeg 自动截取视频第 5 秒画面，CSS cover 填充
- **断点续传** — 服务端 .chunks 文件 + 浏览器 localStorage 双端记录
- **版权声明弹窗** — 首页打开时弹出侵权通知，附联系邮箱
- **评论系统** — 昵称+内容发表，@回复，蜜罐防 spam，管理员可删除
- **中文文件名** — URL 自动 encodeURIComponent 编码

## 部署流程

### 前端更新
```bash
git add .
git commit -m "更新说明"
git push
# Vercel 自动部署，等待约 30 秒
```

### 后端更新（腾讯云服务器）
```bash
ssh ubuntu@124.223.88.48
cd /home/ubuntu/NetDisk
# 修改 src/HttpApiServer.cpp
g++ -std=c++11 -I./include -I/usr/include/mysql src/HttpApiServer.cpp \
    -lpthread -lmysqlclient -lmicrohttpd -ljson-c -o HttpApiServer
kill $(pgrep -x HttpApiServer)
nohup ./HttpApiServer &
```
