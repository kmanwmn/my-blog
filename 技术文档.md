## 后端 API 文档

### 基础地址
- 开发: `http://127.0.0.1:5000`
- 生产: 通过 Vercel 代理 `https://sensing-origin.cn/api/*`

### 视频管理

#### 获取视频列表
```
GET /api/blog-videos
```
返回所有视频，按 sort 降序排列。

#### 获取单个视频
```
GET /api/blog-videos/:id
```

#### 新建视频（需登录）
```
POST /api/blog-videos
Content-Type: application/json
Authorization: Bearer <token>

{
    "title_zh": "视频标题",
    "desc_zh": "视频描述",
    "filename": "blog_xxx.mp4",
    "duration": "03:45",
    "sort": 0,
    "cover_gradient": "linear-gradient(135deg,#0a1628,#1a3a5a,#0d2840)",
    "cover_icon": "🎬"
}
```

#### 编辑视频（需登录）
```
PUT /api/blog-videos/:id
```

#### 删除视频（需登录）
```
DELETE /api/blog-videos/:id
```

### 评论管理

#### 获取评论列表
```
GET /api/blog-comments?video_id=1
```
返回该视频的所有评论，按时间升序。

#### 发表评论（无需登录）
```
POST /api/blog-comments
Content-Type: application/json

{
    "video_id": 1,
    "nickname": "访客名",
    "email": "可选@邮箱.com",
    "content": "评论内容",
    "parent_id": 0,
    "honeypot": ""
}
```
- `honeypot` 为蜜罐防 spam 字段，留空即可，被填时后端静默返回成功但不保存
- `parent_id` 回复时填被回复评论的 ID，首次评论填 0

#### 删除评论（需登录）
```
DELETE /api/blog-comments/:id
```

### 认证

#### 管理员登录
```
POST /api/auth
Content-Type: application/json

{"username": "liyuchen", "password": "colin123"}
```

返回 token，后续请求在 `Authorization: Bearer <token>` 中使用。

### 视频上传（需登录）

```
POST /api/upload/blog-video
Content-Type: application/octet-stream
Authorization: Bearer <token>
X-Filename: demo.mp4
```

支持分片上传，URL 参数加 `?chunk=0&total=3`。
文件自动添加 `blog_` 前缀保存（如 `blog_demo.mp4`），避免与感知起源视频冲突。

## 数据库表结构

### t_BlogVideos

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT AUTO_INCREMENT | 主键 |
| sort | INT | 排序 |
| title_zh | VARCHAR(200) | 标题 |
| title_en | VARCHAR(200) | 英文标题 |
| desc_zh | TEXT | 描述 |
| desc_en | TEXT | 英文描述 |
| filename | VARCHAR(280) | 文件名 |
| duration | VARCHAR(20) | 时长 |
| cover_gradient | VARCHAR(200) | 封面渐变 CSS |
| cover_icon | VARCHAR(20) | 封面图标 emoji |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### t_BlogComments

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT AUTO_INCREMENT | 主键 |
| video_id | INT | 关联视频 ID |
| parent_id | INT | 回复的父评论 ID（可为 NULL） |
| nickname | VARCHAR(50) | 昵称 |
| email | VARCHAR(200) | 邮箱（可选） |
| content | TEXT | 评论内容 |
| created_at | TIMESTAMP | 创建时间 |

## 前端特性

- **暗色/浅色主题切换** — 右上角按钮，设置存入 localStorage
- **响应式布局** — 适配手机/平板/桌面
- **管理员面板** — Ctrl+` 快捷键打开，支持视频 CRUD 和分片上传
- **评论系统** — 昵称+内容发表，@回复，蜜罐防 spam，管理员可删除

## 部署流程

### 前端更新
```bash
git add .
git commit -m "更新说明"
git push
# Vercel 自动部署，等待约 30 秒
```

### 后端更新（腾讯云服务器）
```bash
ssh ubuntu@124.223.88.48
cd /home/ubuntu/NetDisk
# 修改 HttpApiServer.cpp
g++ -std=c++11 -I./include -I/usr/include/mysql src/HttpApiServer.cpp -lpthread -lmysqlclient -lmicrohttpd -ljson-c -o HttpApiServer
kill $(pgrep -x HttpApiServer)
setsid ./HttpApiServer &>/tmp/api.log &
```
